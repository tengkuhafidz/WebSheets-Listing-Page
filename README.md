# SheetySite

This is a Gatsby template for creating a website based on Google sheets.

## Underlying Tech

- Typescript
- TailwindCSS
- Sheets API

I have also configured the following setup and configurations, such as:

- SEO
- PWA and offline capabilities
- Site metadata
- eslint & prettier
- husky
